Documentação da API

Integre seu sistema para gerar cobranças via PIX e automatizar pagamentos em tempo real

● API operacional
Autenticação
Utilize sua chave API no header das requisições: 
x-api-key: SUA_CHAVE_API
Criar cobrança
Endpoint para gerar pagamentos via PIX. 
POST /v1/transactions

{
  "amount": 10000,
  "description": "Pedido #123"
}
Resposta
Retorno da API após criação da cobrança: 
{
  "transactionId": "abc123",
  "pix_code": "000201...",
  "status": "PENDING"
}
Status do pagamento
PENDING PAID FAILED
Webhook
Receba notificações automáticas quando o pagamento for confirmado. 
{
  "event": "payment_paid",
  "transactionId": "abc123",
  "status": "PAID"
}
Regras financeiras
• Taxa por saque: R$ 9,90
• Saque mínimo: R$ 119,90
• Limite por saque: R$ 1.000,00
• Saques ilimitados disponíveis
Fluxo do sistema
1. Operador gera cobrança
2. Cliente realiza pagamento PIX
3. Status inicia como PENDING
4. Webhook atualiza para PAID
5. Saldo é liberado automaticamente
6. Operador pode solicitar saque
